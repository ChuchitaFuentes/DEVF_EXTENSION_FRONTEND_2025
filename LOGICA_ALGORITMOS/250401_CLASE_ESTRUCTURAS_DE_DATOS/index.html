Estructura de datos

Una estructura de datos es una manera de organizar, manejar y almacenar datos que puedan utilizarse de manera eficiente. Cada estructura ofrece diferenter métodos para realizar operaciones como inserción, eliminación búsqueda y acceso a los datos. Se suelen utilizar más en backend.

Lineales:

  • Arreglos:Almacena un conjunto de elementos, bajo un nombre en común, se organizan de manera contigua en memoria, y se accede a ellos utilizando un índice o subíndice.
  • Pilas: Es una estructura que sigue el principio LIFO(Last In, First Out), lo que significa que el último elemento que se agega es el primero en ser eliminado. Operaciones básicas: push, pop, peek o top, isEmpty
  • Colas: Una cola es de tipo (FIFO) (First IN First Out), lo que significa que el primer elemento que se agrega es el primero en ser eliminado. Sus operaciones basicas: Enqueeue(), dequeue, front o peek(), isEmpty()
